How to create BitmapFont(font.bytes, font_texture.png).

1. Execute fontconv.rb. https://github.com/gree/lwf, tools/fontconv/fontconv.rb
  $ macruby fontconv.rb [fontFile] [fontName] [Size] "" [TextureWidth] [TextureHeight] font

2. Import font_texture.png.
Texture Inspector
  Texture Type: Advanced
  Generate Mip Maps: NO
  Max Size: 2048
  Format: Alpha 8

--
How to generate the sample data.
  $ macruby fontconv.rb Tuffy_Bold_Italic.ttf Tuffy 50 "" 2048 2048 font

--
